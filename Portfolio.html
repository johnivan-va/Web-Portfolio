My School Projects

Matching Game

A fun and interactive memory matching game developed using Android Studio. The game challenges players to match pairs of hidden cards, improving memory and concentration skills. Built with Java and XML, it features smooth animations, a responsive layout, and tracks the number of attempts made. Ideal for casual users and educational use.

Women with Baskets

A digital showcase project created using VSCode that highlights Anita Magsaysay-Ho’s iconic painting Women with Baskets. This project presents a visually engaging display of the artwork’s history, style, and cultural significance, combining clean design and responsive layout. Ideal for art enthusiasts and educational purposes, it brings Filipino modern art to the web in an accessible format.

ARNGREN.net

A web project inspired by the unique and chaotic design style of Arngren.net, known for its dense and colorful layout filled with a wide variety of products and links. Built using HTML, CSS, and JavaScript in VSCode, this project explores unconventional web design by recreating Arngren.net’s distinctive look while improving usability and navigation. Perfect for experimenting with bold UI concepts and complex content organization.
